# Curadoria para se manter engajado no mundo tech
 ----
Este repo reúne uma variedade de recursos para quem deseja ficar engajado no mundo da tecnologia. Aqui você encontrará links para comunidades online, canais do YouTube que abordam temas relevantes, podcasts sobre tecnologia, cursos, comunidades, além de outros recursos úteis. Se você está buscando uma forma de se manter atualizado e envolvido no mundo da tecnologia, este repositório é um bom lugar para começar!!
